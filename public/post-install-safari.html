Safari website access setup

Consumer Rights Wiki checks the current page to find matching articles. In Safari, that works best when the extension is allowed to access All Websites.

  1. Open Safari settings.
  2. Select the Extensions tab.
  3. Choose Consumer Rights Wiki in the sidebar.
  4. Find the website access setting and change it to All Websites.

Tip

If Safari asks for permission on a page first, approve it and then switch the extension to All Websites from Safari settings.
