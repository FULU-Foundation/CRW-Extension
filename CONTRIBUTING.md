## I Have a Question

If you want to ask a question, please first:  
- Read the available [documentation]().  
- Search the [issue tracker](https://github.com/FULU-Foundation/CRW-Extension/issues) for existing answers.  
- Search the internet to see if the question has already been discussed elsewhere.  

If you still need clarification:  
- Open a new [issue](https://github.com/FULU-Foundation/CRW-Extension/issues/new).  
- Provide as much context as you can, including your environment (OS, browser, project setup).  

We will review the issue as soon as possible.

## I Want To Contribute

Check out the [project board](https://github.com/FULU-Foundation/CRW-Extension/projects) to see what’s in progress and what still needs doing.  
You can also look at issues labelled **good first issue** for beginner-friendly tasks.  

Pick something that interests you, comment on the issue to let others know you’re working on it, and then open a Pull Request when you’re ready.

### Reporting Bugs

#### Before Submitting a Bug Report

To help us fix issues quickly, please:  
- Make sure you’re using the latest version.  
- Confirm the problem is not caused by your own setup (e.g. incompatible versions).  
- Check the [issue tracker](https://github.com/FULU-Foundation/CRW-Extension/issues?q=label%3Abug) for existing reports.  
- Search online to see if the issue has been discussed elsewhere.  

When preparing your report, collect the following:  
- Browser and version (e.g. Chrome, Firefox)  
- Operating system and version  
- Steps to reproduce (what you did, what you expected, what actually happened)  
- Any errors from the console or logs  
- Project setup details 
- Whether the issue is reproducible and if it occurs across multiple browsers  

#### How Do I Submit a Good Bug Report?

⚠️ **Do not report security issues publicly.** Send them directly to <jake@fulu.org>.  

For other bugs:  
- Open an [issue](https://github.com/FULU-Foundation/CRW-Extension/issues/new).  
- Clearly describe the expected vs actual behavior.  
- Provide reproduction steps so others can replicate the issue.  
- Include the information you collected above.  

Once filed:  
- The project team will label the issue appropriately.  
- A team member will try to reproduce it. If steps are missing, the issue may be marked `needs-repro` until it can be reproduced.  
- Verified bugs will be marked `needs-fix` (and other relevant tags) and left open for implementation.

### Suggesting Enhancements

Enhancement suggestions are tracked as [GitHub issues](https://github.com/FULU-Foundation/CRW-Extension/issues).  

Before suggesting an enhancement:  
- Make sure you’re using the latest version.  
- Read the [documentation]() to confirm it isn’t already supported.  
- Search the issues to avoid duplicates and comment on an existing issue if it already exists.  

When opening an enhancement issue, please:  
- Use a clear and descriptive title  
- Explain the current behavior and the change you’d like to see  
- Describe why the change would be useful  
- Add screenshots or examples if they help illustrate the idea
